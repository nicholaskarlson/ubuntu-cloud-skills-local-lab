# Ubuntu Cloud Skills for Independent Creators

Working series plan for NEKpress.ca.

## Book 1

Your First Local Cloud Lab with Ubuntu and Docker

Learn servers, websites, logs, files, and safe deployment habits without paying for a cloud account.

## Book 2

Your First Ubuntu Cloud Server

Move the same ideas to a real VPS using SSH, users, updates, firewall rules, domains, HTTPS, and backups.

## Book 3

Publish Your Creator Website on Ubuntu

Deploy a practical static or lightly dynamic creator website using Git, Caddy or Nginx, logs, and repeatable updates.

## Book 4

Build a DIY Digital Bookstore

Create product pages, protected download workflows, order records, and fulfillment habits.

## Book 5

Stripe Checkout for Independent Creators

Use hosted checkout, test mode, webhooks, receipts, and safer digital delivery.

## Book 6

AWS Skills for Ubuntu Creators

Use AWS CLI, IAM, S3, free-tier-aware practice, receipts, and cost-safe cleanup.
